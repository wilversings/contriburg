# Contriburg — Your Contribution Graph, in 3D 🏙️

Your flat little contribution graph deserves better. Contriburg turns it
into a 3D skyline right on your desktop — every day a cube, its height and
color a monument to how much you shipped (or didn't 👀). Orbit it, zoom
it, hover a cube for the receipts.

---

## ✨ Features

- GitHub or GitLab as the data source (pick a side)
- 52×7 grid of cubes, scaled and colored by contribution level
- Orbit/zoom camera, or lock it in place if you're not feeling adventurous
- Per-cube hover tooltips, because everyone loves receipts
- Fixed accent color, or let every cube roll its own random hue 🎲
- Transparent background — blends in, doesn't show up in a box uninvited

---

## ⚙️ Configuration

Right-click the widget → **Configure...** for username, data source,
refresh interval, cube height/color, and camera lock.

---

## ⚠️ Requirements

- KDE Plasma 6.0+
- **Qt Quick 3D** installed — `qt6-qtquick3d` (Fedora), `qt6-3d` (Arch),
  `libqt6qtquick3d6` (Debian/Ubuntu), `libqt6qtquick3d` (openSUSE)
- A GitHub Personal Access Token is optional (unlocks the GraphQL API) —
  everything else runs happily token-free on public profiles. 🔓
